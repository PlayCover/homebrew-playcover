# homebrew-playcover

Homebrew cask for PlayCover

## Casks
```
playcover-community
playcover-prerelease
playcover-nightly
```

## Installing

```
brew tap playcover/playcover
brew install --cask playcover-community
```

## Uninstalling

```
brew uninstall --cask playcover-community
brew untap playcover/playcover
```
